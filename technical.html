Picturing the Power of Policy in a Pandemic - Python

In this project, we are interested in analyzing the efficacy of various global policy responses in reducing deaths and cases. In doing so, we hope to learn more about the interconnections between policies and their role in reducing transmission of COVID-19, and shed light on the policy response(s) that have been most effective in their respective regions. This project uses Python as well as libraries like plotly, networkx and pandas. This project is a collaborative work with Komal Saini as part of CSC111 Final Project, where we have to use graphs and graphs algorithm to represent our data.

Vaxman - Python

Vaxman is a slightly modified version of the game "Pacman", where vaxman can kill a ghost if he comes into contact with it (but does not kill vaxman). Each ghost that has not been killed multiplies itself every 30 seconds, and the goal of the game is to collect all the dots before the number of ghosts grows to 32 times the original number. This project is made using Python and pygame library.
